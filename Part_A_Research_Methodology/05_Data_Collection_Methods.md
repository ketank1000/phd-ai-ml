# 5. Data Collection Methods

> **Part A — Research Methodology** | SIU PET Study Guide

---

- [ ] **Primary Data Collection**
  - **Questionnaire**: Structured, semi-structured, unstructured; Likert scale
  - **Interview**: Structured, unstructured, in-depth, focus group
  - **Observation**: Participant, non-participant, structured, unstructured
  - **Experiment**: Laboratory, field experiment

- [ ] **Secondary Data Collection**
  - Published sources: journals, books, government reports
  - Unpublished sources: theses, company records
  - Electronic sources: databases, web resources

- [ ] **Scales of Measurement**

  | Scale | Properties | Examples | Allowed Stats |
  |-------|-----------|----------|--------------|
  | **Nominal** | Categories only | Gender, color | Mode, frequency |
  | **Ordinal** | Order/rank | Rating (1st, 2nd..), Likert | Median, percentiles |
  | **Interval** | Equal intervals, no true zero | Temperature (°C), IQ scores | Mean, SD, correlation |
  | **Ratio** | Equal intervals + true zero | Height, weight, accuracy % | All statistical operations |

- [ ] **Data Quality**
  - Accuracy, completeness, consistency, timeliness
  - Data cleaning, handling missing values
  - Triangulation: using multiple data sources/methods
