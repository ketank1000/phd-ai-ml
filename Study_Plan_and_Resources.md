# Study Plan & Exam Strategy

> **SIU PhD Entrance Test (PET) 2025** | Computer Engineering — AI/ML Specialization

## Priority Order (by expected weightage for AI/ML specialization)

| Priority | Topics | Estimated Study Time |
|----------|--------|---------------------|
| **P0 (Critical)** | ML Supervised (§19), Deep Learning (§21), Math for ML (§18) | 35% of time |
| **P1 (High)** | Research Methodology (§1-8), NLP (§22), Unsupervised ML (§20) | 25% of time |
| **P2 (Important)** | DSA (§9), DBMS (§10), Computer Vision (§23), OS (§11) | 20% of time |

---

## Week-by-Week Study Plan (8-Week Plan)

### Week 1-2: Foundations
- [ ] Research Methodology (Chapters 1-8 — Part A complete)
- [ ] Mathematical Foundations for ML (Linear Algebra, Probability, Calculus)

### Week 3-4: Core ML
- [ ] Supervised Learning (all algorithms, evaluation metrics)
- [ ] Unsupervised Learning (clustering, dimensionality reduction)
- [ ] Practice ML problems and coding exercises

### Week 5: Deep Learning
- [ ] Neural network basics, backpropagation
- [ ] CNN architectures
- [ ] RNN, LSTM, GRU
- [ ] Transformers & Attention

### Week 6: NLP + Computer Vision
- [ ] NLP fundamentals and advanced topics
- [ ] Computer Vision fundamentals and applications
- [ ] Reinforcement Learning basics

### Week 7: Core CS
- [ ] Data Structures & Algorithms (focus on complexity, key algorithms)
- [ ] DBMS (SQL, normalization, transactions)
- [ ] Operating Systems (scheduling, memory management, deadlock)
- [ ] Computer Networks, TOC, Discrete Math (revision)

### Week 8: Revision & Mock Tests
- [ ] Revise all formulas and key concepts
- [ ] Solve previous year papers (if available)
- [ ] Take timed mock tests
- [ ] Review weak areas
