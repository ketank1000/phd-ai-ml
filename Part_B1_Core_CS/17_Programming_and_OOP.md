# 17. Programming & OOP Concepts

> **Part B1 — Core Computer Science** | SIU PET Study Guide

---

- [ ] **OOP Principles**
  - **Encapsulation**: Data + methods bundled together; access modifiers (public, private, protected)
  - **Inheritance**: Code reuse through parent-child relationship; types (single, multi-level, hierarchical, multiple via interfaces)
  - **Polymorphism**: Compile-time (method overloading) vs. Runtime (method overriding/virtual functions)
  - **Abstraction**: Hiding implementation details; abstract classes, interfaces

- [ ] **Key Programming Concepts**
  - Pointers and references (C/C++)
  - Memory management: stack vs. heap, garbage collection
  - Recursion vs. iteration
  - Exception handling: try-catch-finally
  - Generics/templates
  - Lambda functions, closures

- [ ] **Python-Specific** (relevant for AI/ML coding)
  - List comprehensions, generators, decorators
  - NumPy, Pandas basics
  - Pass by object reference
  - GIL (Global Interpreter Lock)
