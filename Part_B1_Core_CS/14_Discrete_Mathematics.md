# 14. Discrete Mathematics

> **Part B1 — Core Computer Science** | SIU PET Study Guide

---

- [ ] **Set Theory**
  - Set operations: union, intersection, difference, complement, Cartesian product
  - Power set, cardinality
  - De Morgan's laws: $(A \cup B)' = A' \cap B'$ and $(A \cap B)' = A' \cup B'$

- [ ] **Relations**
  - Properties: reflexive, symmetric, transitive, antisymmetric
  - Equivalence relation: reflexive + symmetric + transitive
  - Partial order: reflexive + antisymmetric + transitive
  - Hasse diagram

- [ ] **Functions**
  - Types: injective (one-to-one), surjective (onto), bijective (both)
  - Composition, inverse functions
  - Pigeonhole principle

- [ ] **Propositional Logic**
  - Operators: AND (∧), OR (∨), NOT (¬), IMPLIES (→), BICONDITIONAL (↔)
  - Truth tables, tautology, contradiction, contingency
  - Logical equivalences, normal forms (CNF, DNF)

- [ ] **Predicate Logic**
  - Universal quantifier (∀), existential quantifier (∃)
  - Predicates, bound and free variables

- [ ] **Graph Theory**
  - Degree of vertex, handshaking lemma: $\sum deg(v) = 2|E|$
  - Euler path (all edges once), Euler circuit
  - Hamiltonian path (all vertices once), Hamiltonian cycle
  - Graph coloring, chromatic number
  - Planar graphs, Kuratowski's theorem, Euler's formula: V − E + F = 2
  - Trees: spanning tree, properties (n vertices → n-1 edges)

- [ ] **Combinatorics**
  - Permutations: $P(n,r) = \frac{n!}{(n-r)!}$
  - Combinations: $C(n,r) = \binom{n}{r} = \frac{n!}{r!(n-r)!}$
  - Binomial theorem: $(x+y)^n = \sum_{k=0}^{n} \binom{n}{k} x^k y^{n-k}$
  - Inclusion-exclusion principle
  - Recurrence relations, generating functions

- [ ] **Number Theory**
  - Divisibility, GCD (Euclidean algorithm), LCM
  - Modular arithmetic: $(a \cdot b) \mod m = ((a \mod m) \cdot (b \mod m)) \mod m$
  - Fermat's little theorem, Euler's totient function
